The architecture of neural networks

input layer -> single hidden layer/multiple hidden layer(MLP) -> output layer

feedforward -> the output from one layer is used as input to the next layer, there are no loops in the network - information is always fed forward.

recurrent neural networks -> is a class of artificial neural network where connections between units form a directed cycle.

Gradient descent

cost function: To quantify how well we’re achieving this goal we define a cost function. So the aim of our training algorithm will be to minimize the cost C（w,b） as a function of the weights and biases. Gradient descent is a method to achieve these goals.

cost function

  • Quadratic cost (MSE/Maximum Likelihood)
  • Cross-entropy cost (Bernoulli negative log-likelihood/Binary Cross-Entropy)
  • Exponentional cost
  • Hellinger distance
  • Kullback-Leibler divergence (Information Divergence/Information Gain)
  • Generalized Kullback - Leibler divergence
  • Itakura - Satio distance

Stochastic gradient descent

The idea is to estimate the gradient ▽C by computing ▽Cx for a small sample of randomly chosen training inputs. By averageing over this small sample it turns out that we can quickly get a good estimate of the true gradient ▽C, and this helps speed up gradient descent, and thus learning.

To make these idears more precise, stochastic gradient descent works by randomly picking out a small number m of randomly chosen training inputs.

mini-batch: randomly picking out a small number m of randomly chosen training inputs. We can estimate the overall gradient by computing gradients just for randomly chosen mini-batch.

epoch: consists of many mini-batchs. Like a year consists of 12 months.
